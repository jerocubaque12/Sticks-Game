Sticks Game 🎯

A simple Python game based on the classic “short stick” challenge.
The player must choose one stick from a hidden shuffled list, and if they pick the shortest stick, they lose.

This project was developed to practice:

Functions
Lists
Loops
User input validation
Randomization with Python
Features
🎲 Random stick shuffling using Python’s random module.
🎮 Interactive terminal gameplay.
✅ Input validation to prevent invalid choices.
📌 Simple and beginner-friendly code structure.
🧠 Practice with functions and list handling.
How to use
Clone this repository:
git clone https://github.com/jerocubaque12/sticks_game.git
Open the project folder.
Run the script:
python sticks_game.py
Choose a number between 1 and 4 and test your luck!
Example Gameplay
Type any number between 1 and 4: 2
You lost!

or

Type any number between 1 and 4: 4
This time you won!, hope you dont have better luck next time
